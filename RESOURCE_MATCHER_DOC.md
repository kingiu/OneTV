# 影视资源精准匹配功能技术文档

## 1. 实现方案概述

本实现基于TypeScript开发了一个影视资源匹配引擎，旨在解决OneTV项目中影视资源匹配不准确的问题。该引擎参考了LunaTV项目的匹配逻辑，并进行了多项优化，提升了匹配准确性和覆盖率。

### 1.1 核心目标

- 改进影视资源元数据的提取与标准化处理
- 优化搜索匹配算法，支持模糊匹配、同义词匹配及多条件组合匹配
- 建立资源匹配质量评估机制
- 确保与现有系统架构兼容

## 2. 核心功能介绍

### 2.1 元数据提取与标准化处理

- **标题标准化**：去除特殊字符、多余空格，进行繁简转换
- **年份提取**：从标题中自动提取年份信息
- **演员/导演信息提取**：从描述中解析演员和导演信息
- **类型信息提取**：从类型字段中解析影视类型

### 2.2 智能搜索变体生成

- **数字变体**：支持中文数字与阿拉伯数字的相互转换（如"第一季" ↔ "第1季"）
- **标点变体**：处理中文冒号、英文冒号、书名号等标点符号
- **空格变体**：处理带空格和无空格的搜索词
- **繁简变体**：自动进行繁简体转换
- **同义词变体**：支持常见同义词的替换

### 2.3 模糊匹配和同义词匹配

- **模糊匹配**：使用Levenshtein距离算法计算字符串相似度
- **同义词匹配**：内置常见影视相关同义词映射表

### 2.4 多条件组合匹配

- **标题匹配**：权重最高，支持完全匹配和部分匹配
- **年份匹配**：验证年份信息的一致性
- **类型匹配**：检查类型信息的匹配度
- **演员/导演匹配**：匹配演员和导演信息
- **结果质量加分**：根据剧集数量给予额外分数
- **来源可靠性加分**：根据来源的可靠性给予额外分数

### 2.5 匹配质量评估机制

- **平均分数**：计算所有匹配结果的平均分数
- **最高分数**：获取匹配结果中的最高分数
- **匹配数量**：统计匹配成功的结果数量
- **覆盖率**：评估匹配的覆盖情况

## 3. 技术架构

### 3.1 核心类与接口

- **ResourceMatcher**：核心匹配引擎类
- **VideoMetadata**：影视资源元数据接口
- **SearchResult**：搜索结果接口
- **MatchResult**：匹配结果接口

### 3.2 数据流

1. 接收搜索请求和搜索结果
2. 提取并标准化每个搜索结果的元数据
3. 生成搜索变体
4. 计算每个结果与搜索词的匹配分数
5. 按分数排序并过滤低质量结果
6. 评估匹配质量
7. 返回优化后的搜索结果

### 3.3 集成方式

- 集成到`services/api.ts`中的`searchVideos`和`searchVideo`方法
- 在搜索结果处理后使用匹配引擎进行优化
- 保持与现有API接口的兼容性

## 4. 优化点分析

### 4.1 算法优化

- **Levenshtein距离算法**：用于计算字符串相似度，比简单的字符串比较更准确
- **多条件加权匹配**：综合考虑多个因素，提高匹配准确性
- **智能搜索变体**：生成多种搜索变体，提高覆盖率

### 4.2 性能优化

- **并行处理**：搜索变体的生成和处理可以并行进行
- **缓存机制**：可以缓存常见搜索词的匹配结果
- **结果过滤**：过滤低分数结果，减少后续处理开销

### 4.3 准确性优化

- **元数据标准化**：统一处理不同来源的元数据格式
- **同义词映射**：处理常见同义词，提高匹配准确性
- **多条件验证**：综合多个条件进行匹配，减少误匹配

## 5. 使用方法

### 5.1 基本用法

```typescript
import { resourceMatcher } from './services/resourceMatcher';

// 处理搜索结果
const query = '流浪地球2';
const searchResults = [...]; // 从API获取的搜索结果

const processed = resourceMatcher.processSearchResults(query, searchResults);
const optimizedResults = processed.matches.map(match => match.result);
```

### 5.2 高级用法

```typescript
// 生成搜索变体
const variants = resourceMatcher.generateSearchVariants('流浪地球2');

// 单独使用匹配功能
const matches = resourceMatcher.matchResources('流浪地球2', searchResults);

// 评估匹配质量
const quality = resourceMatcher.evaluateMatchQuality(matches);
```
